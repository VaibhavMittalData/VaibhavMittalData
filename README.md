## 🚀 Projects

### 🤖 AI Mood-Based Chatbot
> *Feb 2026 – Mar 2026*

An interactive AI chatbot with **Angry, Funny & Sad** personalities powered by LLMs.
- Integrated **LangChain** with **Groq's LLaMA-3.3-70B** for real-time responses
- Session-based memory to maintain conversation context
- **Tech Stack:** `Python` `Streamlit` `LangChain` `Groq API`

---

### 🚗 Car Accident Detection (CNN)
> *Jan 2026 – Feb 2026*

Binary image classification model to detect car accidents using deep learning.
- Custom CNN architecture with **92% accuracy**
- Full ML pipeline: preprocessing → augmentation → training → evaluation
- **Tech Stack:** `Python` `PyTorch` `Torchvision` `Scikit-learn`

---

### 🏥 Cancer Prediction Model
> *Oct 2025 – Nov 2025*

ML model using clinical & lifestyle data to identify high-risk cancer patients.
- EDA, preprocessing, multi-model evaluation, and Streamlit deployment
- **Tech Stack:** `Python` `NumPy` `Pandas` `Scikit-learn` `Matplotlib` `Streamlit`
